# Order Inventory Management System

Spring Boot backend API with JWT authentication, refresh tokens, and inventory management.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- MySQL
- Swagger / OpenAPI
- JUnit & Mockito

## Features
- Authentication & Authorization
- Product & Inventory APIs
- Role-based access control
- Global exception handling
